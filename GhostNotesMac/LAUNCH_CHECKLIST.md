# GhostNotesMac — Launch Checklist

## Pre-Launch

### App Store Assets
- [ ] App Store screenshots (5.5" and 6.7" for iOS, 12.9" for iPad, macOS screenshot)
- [ ] App Store icon (1024x1024 master)
- [ ] Tagline finalized: "Read deeply. Remember everything."
- [ ] Description written and reviewed
- [ ] Keywords researched and populated
- [ ] Privacy policy URL ready
- [ ] Support URL ready
- [ ] Marketing website landing (optional)

### Code & Quality
- [ ] Build succeeds (Release, Apple Silicon)
- [ ] No warnings in Release build
- [ ] App Store Connect provisioning
- [ ] App Store Connect bundle ID matches
- [ ] Version number & build number correct
- [ ] Minimum macOS version set appropriately
- [ ] Entitlements configured (if needed)

### Accounts & Access
- [ ] Apple Developer account active
- [ ] App Store Connect app created
- [ ] App Store Connect agreements current (Paid Apps)
- [ ] Bank account / payment info set up
- [ ] Tax information complete

### Launch Strategy
- [ ] Release date set
- [ ] Phased release or full release chosen
- [ ] Pricing: Free / Free with IAP / Paid ($X.99)

---

## Launch Day

### Submission
- [ ] Xcode build uploaded to App Store Connect
- [ ] All screenshots uploaded
- [ ] Metadata filled in
- [ ] Privacy policy URL included
- [ ] App Review submitted

### Monitoring
- [ ] App Store Connect status: "In Review"
- [ ] Email notifications enabled for review status
- [ ] Prepare for potential rejection & fix cycle

---

## Post-Launch

### Day 1–7
- [ ] Monitor App Store Connect for approval
- [ ] Check for crashes (Xcode Organizer / Crashlytics)
- [ ] Monitor review feedback if rejected

### Week 1–2
- [ ] First user feedback reviewed
- [ ] Ratings & reviews responded to
- [ ] App Store ranking tracked
- [ ] Social announcement (if planned)

### Ongoing
- [ ] Bug fixes as reported
- [ ] Feature updates planned
- [ ] User support handled
